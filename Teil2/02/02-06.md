## NAS ( Network Attached Storage)
Ein Speichermedium welches über das Internet zugegriffen werden kann.

## SAN (Storage Area Network)
Ein Netzwerk voller speichermedien die vom Netzwerk zugegiffen werden können (NAS)

## iSCSI (Internet Small Computer System Interface)
iSCSI ist eine Netzwerkprotokoll wie SNMP, HTTP usw welches zugriff auf Speichermedien im Netzwerk ermöglicht.

## SMB (Server Message Block)
Netzwerkprotokoll welches den Dateizugriff, Dateidruck und zugriff zu verschieden anderen Serverdiensten ermöglicht.
## NFS (Network File System)
NFS ermöglicht Dateizugriff auf externen Computern. Im gegenteil zu FTP, welches grundsätzlich das gleiche Absolviert, verhält sich die externe Festplatte als ob sie an dem PC vom holer der Dateien befindet.

## Ethernet
Ermöglicht Datenübertragung in lokalen Netzwerken.
