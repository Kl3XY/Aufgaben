## On Premises
Server Geräte und so weiter befinden sich in den eigenen Räumlichkeiten der Firma
## Cloud
Server Geräte, Infrastruktur wird von einem Remote Ort zugegriffen. 
## Software as a Service
Programme werden über das Internet für den Endnutzer bereitgestellt.
## Infrastructure as a Service
Dem Endnutzer werden Server, Speicher etc. zu verfügung gestellt
## Platform as a Service
Ein ort der gemacht wurde für das Bauen, Deployen und managen der entwickelten Anwendungen.
