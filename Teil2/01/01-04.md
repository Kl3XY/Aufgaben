## SMB (Server Message Block-Protokoll) bzw. CIFS
SMB ist ein Protokoll was den Zugriff auf Ordner, Dateien und Drucker im Netzwerk ermöglicht.
Hierbei schickt der Nutzer eine Anfrage Antwort an den Server um die darauf hinterlegenden Daten zuzugreifen. Der Server, nach authentifizierung des Clients, gibt die Ressourcen frei.
#### - NTLM (NT Lan Manager)
Bei dem Authentifizierungsprozess wird NLTM am Haüfigsten benutzt hierbei wird anhand der Windows Anmeldedaten Credentials dem Nutzer zugang zu dem Server erlaubt.

## HTTP
HTTP ist die Grundlage im World Wide Web (oder Internet) zugriff auf Webseiten und Kommunikation der einzelnen clients zu ermöglichen.

## ODBC
Eine, von Microsoft, bereitgestellte API die, die Kommunikation zwischen dem meisten Datenbankmanagemensystem und dem Nutzer (Client), ermöglicht.
