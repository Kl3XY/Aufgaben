## DNS (Domain Name System)
Soziemlich das Telefonbuch des Internets. Hiermit ist es möglich IP-Addressen namen zu geben. Damit ist es einfacher sich IP-Adressen zu merken. 

z.b 13.10.2025:

193.99.144.80 = heise.de
81.169.145.7 = minecraft.de
142.250.186.110 = youtube.de

## DHCP (Dynamic host Configuration Protocol)
Hiermit werden über den DHCP-Dienst, der auf einem Server gehostet wird, Adressen automatisch vermietet.

## Proxy
Eine art maske für Verbindnungen. In richtig zusammengefasst ist es ein Mittelman Server zwischen der Absender und Empfänger. Hierbei ist das Ziel den Absender zu verdecken um die Verbindung scheinen zu lassen dass es von irgendwo anders kommt.
