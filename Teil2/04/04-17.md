## Kapselung
Kapselung beschreibt es manche Variabeln usw. unerreichbar von aussen zu machen indem man sie auf "private" stellt.
```
using System;
namespace ShowcaseProgramm {
	public class Core() {
		private string _version = "1.0"
		public string Version 
		{ 
			get {
				return _version;	
			} 
		}
	}
}
```
Dieser code sorgt dafür dass der eigentlich Version string niemals von aussen bearbeitet werden kann sonder nur durch geholt werden kann mit dem custom Get Setter
## Vererbung
Vererbung ist mit einer der Wichtigsten Prinzipien von der Objektorientiereten Programmierung. wo eine Klasse von einer andere dessen Methoden und Variabeln vererben kann.
```
using System;
namespace ShowcaseProgramm {
	public class Car() {
		public string CarName = "No Car Name"
	}
	
	public class Audi() : Car() {
		public string CarName = "Audi"
	}
}
```

## Polymorphie
Polymorphie beschreibt das ändern eine Methode von der Vererbten klasse. Beispiel hierbei wäre.
```
using System;
namespace ShowcaseProgramm {
	public class Car() {
		public string CarName = "No Car Name"
		
		public void drive() {
			Console.WriteLine("vroom")
		}
	}
	
	public class Mephistopholes() : Car() {
		public string CarName = "Mephistopholes"
		
		public void drive() {
			Console.WriteLine("vroom vroom")
		}
	}
}
```
## Interfaces
Interfaces sind grundsätzlich nur unfertige klassen die so gebaut werden das sie vererbt und erweitert werden
```
namespace ShowcaseProgramm {
	public class ICar() {
		public abstract void Drive();
	}
	
	public class Mephistopholes() : ICar() {
		public abstract void Drive() {
			Console.WriteLine("vroom vroom")
		}
	}
}
```
Hiermit gewährleistest du auch das jede methode die "ICar" benutzt auch garantiert damit funktioniert.
