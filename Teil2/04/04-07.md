## Editor
Ein simpler Text-Editor wie Notepad, Word oder was auch immer. Sie haben kein wirklichen nutzen in der Programmierwelt bisauf dem einblicken in JSON, XML's etc.
## IDE
Integrated Developer Environments sind eine stufe über Editoren und sind spezifisch fürs Programmieren entwickelt worden und bieten typische features wie Highlighting, Intelligentes vervollständigen, KI-Features.
## Programmgenerator
Heutzutage ist es möglich mithilfe von KI-Werkzeugen ganze Programme generative entwickeln lassen. Wenn das ist was es meinst
## Linker
Ein Linker verbindet die Jeweiligen komponente und Bibliothkeen eines Programmes beim Bau- und Kompilierungsprozess.
## Compiler
Compiler geht durch den gesamten Code in deiner Solution und verwandelt diesen in Maschinen code welcher dann zu einer Executeable (.exe, und was auch immer Linux und Mac benutzen) verwandelt wird der dann von deinem CPU normal gelesen werden kann.
## Interpreter
Interpreter übersetzt den Code live während des Ablaufens ins Maschinencode. 
## Debugger
Debugger sind mächtige Werkzeuge die es ermöglichen ein Snapshot eines Programmes an einer Beliebigen stelle eines Codes zu stoppen. Sobald ein Programm gestoppt wurde ist es möglich den jetzigen Stand der Software anzusehen und jede Variabel zu erblicken. Es ist ab dem Punkt möglich die Software schrittweise durchzugehen 
## Testsoftware
Software die pin-point gemacht wurde um Programme zu testen auf verschiedene weisen.
## Versionsverwaltung
Versionsverwaltungssoftware (wie z.B. Git) dokumentiert und behält den Verlauf der Versionen von der Software woran die Versionsverwaltungssoftware angeklebt wurde.
Versionsverwaltungssoftware basiert auf Repositories, in den meisten fällen gibt es ein Lokales und ein Remote Repository. Das Lokale befindet sich immer in deinem PC während das Remote sich auf einem Server meistens befindet. Ein Repository beeinhält den Quellcode der jetzigen version sowie auch die Historie der vergangen Versionen.

Es ist meist ne gute idee ein VVS zu haben einfach nur um Fuck ups zu vermeiden.
