## CREATE TABLE
Erstellt eine Tabelle
```
CREATE TABLE Users(name varchar(30), password varchar(30))
```
## ALTER TABLE
Fügt eine Spalte zu einer Tabelle hinzu
```
ALTER TABLE Users
ADD alias varchar(10)
```
## INDEX
Erstellt eine Index zu einer Tabelle um bestimmte werte mithilfe des neu gesetzten index schneller zu finden.
```
CREATE INDEX
ON Users(alias)
```
## INSERT
fügt ein wert hinzu
```
INSERT INTO Users(name, password, alias)
VALUES ('Martin Gaffke', 'changeme', 'Kl4XY')
```
## UPDATE
Verändert ein wert in einer tabelle
```
UPDATE Users
SET alias = 'Kl3XY'
WHERE alias == 'Kl4XY'
```

## DELETE
Löscht ein wert in einer Tabelle
```
DELETE FROM Users
WHERE alias = 'Kl3XY'
```

## SELECT
```
SELECT * FROM Users WHERE alias == 'Kl3XY'
```

## GROUP BY
```
SELECT * FROM Users WHERE alias LIKE 'K%'
ORDER BY Wallet
```

## HAVING
```
SELECT COUNT(GameID), alias FROM Users
GROUP BY GameID
HAVING COUNT(GameID) > 5;
```

## Abfrage über mehrere Tabellen
```
SELECT u.alias as 'Username', g.name as 'Game'
FROM Users u
INNER JOIN UserGames ug ON ug.userID == u.ID
INNER JOIN Games g ON ug.gameID == g.ID;
```
