## Lineare Suche
Die Lineare suche ist somit die simpelste art einer suche da es das gesamte array durch geht was es durchsuchen soll und nach dem Wert sucht.
```
public bool LinearSearch(int searchNumber, int[] array) {
	bool searchSuccessful = false;
	
	foreach (int item in array) {
		if (item == searchNumber) {
			searchSuccessful = true;
		}
	}
	
	return searchSuccessful;
}
```

## Binäre Suche
Binäre suche basiert auf ein Sortiertes Array. die Binäre such nimmt ein Wert aus der Mitte und vergleich diesen mit dem wert der gesucht wird. Wenn der wert der gesucht wird nicht der mitte entspricht wird geguckt ob der wert größer oder kleiner als der wert ist. wenn er größer ist, halbier das array und nutze die rechte hälfte. Wenn kleine halbier das Array und nutze die linke hälfte. Dieser prozess geht solange weiter bis der wert gefunden wurde.

```
public bool LinearSearch(int searchNumber, int[] array) {
	if (array.length == 0) {
		return false;
	}
	
	bool searchSuccessful = false;
	int mid = array.length/2
	
	if array[mid] == searchNumber {
		return true;
	} else {
		if array[mid] < searchNumber {
			return LinearSearch(searchNumber, array.copy(mid, array.length))
		} 
		else
		{
			return LinearSearch(searchNumber, array.copy(0, mid))
		}
	}
	return false;
	
}

## Bubble Sort
Bubble sort. Du vergleichst zwei werte, wenn der eine Kleiner ist als der andere. swappe die werte.
```
public bool BubbleSort(int[] array) {
	for (var i = array.length; i > 1; i--) {
		for (var j = 0; j < i - 1; i++) {
			if (array[j] > array[j+1]) {
				var temp1 = array[j];
				var temp2 = array[j+1];
				array[j] = temp2;
				array[j+1] = temp1;
			}
		}
	}
}
```

## Selection Sort
Selection Sort nimmt sich den Kleinsten wert des Array und setzt in an erster Stelle und so weiter. Er nimmt sich den nächsten kleinen wert und legt ihn an zweiter stelle und so weiter und so fort

(Source code von geeksforgeeks)
```
static void selectionSort(int[] arr){
        int n = arr.Length;
        for (int i = 0; i < n - 1; i++) {

            // Assume the current position holds
            // the minimum element
            int min_idx = i;

            // Iterate through the unsorted portion
            // to find the actual minimum
            for (int j = i + 1; j < n; j++) {
                if (arr[j] < arr[min_idx]) {

                    // Update min_idx if a smaller element
                    // is found
                    min_idx = j;
                }
            }

           // Move minimum element to its
           // correct position
           int temp = arr[i];
           arr[i] = arr[min_idx];
           arr[min_idx] = temp;         
        }
    }
```

## Insertion Sort
Vergleiche den zweiten wert des Arrays mit dem ersten wert wenn dieser kleiner ist, swap sie. Danach vergleiche den dritten wert mit den letzten zwei und leg ihn dann in den richtigen spot.

Der Algorithmus selbst drückt die werte im bereits sortierten teil so weit nach rechts bis da platz für den wert gemacht wurd.
```
public bool InsertionSort(int[] array) {
	for (var i = 1; i < array.length; i++) {
		var current_value = array[i];
		var j = i - 1;
		
		while (j >= 0 && arr[j] > current_value) {
			arr[j + 1] = arr[j];
			j = j - 1;
		}
		arr[j + 1] = current_value;
	}
}
