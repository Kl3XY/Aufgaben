## Produktivität
Relation zu output und input. Wie viel mehr output kriegst du als dein Input.
```
Produktivität = Output / Input
```

## Wirtschaftlichkeit
Wie Effizient ein Unternehmen ist. Die Wirtschaftlichkeit verwendet das Maximal- und Minimalprinzip

```
Wirtschaftlichkeit = Ertrag / Aufwand
```
#### Minimalprinzip
Das gewünschte Ziel mit so wenig Mitteln wie möglich zu erreichen
#### Maximalprinzip
Das meiste aus dem gegebenen Mitteln machen.
## Rentabilität
Verhältnis zwischen dem Gewinn und dem eingesetzten Kapital
```
Rentabilität = Gewinn / Kapital
```

## Zielkonflikte
Ziele die sich gegenseitig in Konflikt stehen. Nachhaltigkeit und Profite etc. etc.
