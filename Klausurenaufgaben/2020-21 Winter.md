## Beispiel für Inner Joins anhand der Aufgabe.

#### Aufgabe 5 (Ja aus irgendeinen grund gibt es fünf aufgaben.)
![[Pasted image 20251119165754.png]]

Mann soll eine Tabelle ausgeben lassen die den Kundennamen mit den gesamt Umsatz ausgibt

```
SELECT k.Kd_Firma, SUM(rp.RgPos * rp.RgPos_Menge) as 'Gesamtumsatz'
FROM KUNDE k
INNER JOIN RECHNUNG r ON k.Kd_ID == r.Rg_KdID
INNER JOIN RECHNUNG_POSITION rp ON rp.RgPos.RgID == Rg_ID
GROUP BY (k.Kd_Firma, k.Kd_ID)
```

Mein grundsätlicher tipp beim schreiben ist es immer das SELECT rechts leer zu lasen bis alle Tabellen variabel namen erstellt worden sind. Damit vermeidest du das du aufm Blatt u wenig platz hast um den Tatsächtliche Select zu schreiben.
