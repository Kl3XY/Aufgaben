## Penetrations-Test
Eine simulierte Cyberattacke auf IT-Systeme die von "ethischen" oder "White Hat" Hacker durchgeführt wird. DIes wird gemacht um das Verhalten der Sicherheitsmaßnahmen eines Servers oder Dienstes zu prüfen.

Software Beispiele:
- Kali Linux
- Metasploit
- Burp Suite
- Nmap
- Wireshark
## Device Security Check
Nur eine normale Prüfung ob das Geräte up-to-date ist, frei von malware und was genau für Geräte am computer verbunden sind.

Software Beispiele:
- MS Defender
- Jamf
- Nessus
- OpenSCAP
- Cisco Secure Endpoint
## Identity & Access Management
Hierbei prüft mann wer auf was zugriff hat.

Software Beispiele:
- Azure Active Directory
- Okta
- Keycloak
## Schwachstellenanalyse
Was für Schwachstellen das Programm hat.

Software Beispiele:
- Nessus
- OpenVAS
- Qualys
